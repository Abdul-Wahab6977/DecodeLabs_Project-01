# AuroraLearn – Developer DashboardAuroraLearn is a modern, responsive, and pixel-perfect single-page Developer Dashboard built entirely with pure, vanilla web technologies. It features a sleek "Aurora Blue" design language optimized for software engineers and technology professionals tracking their learning paths.The application allows users to search, filter, and sort their active and completed courses effortlessly, complete with sleek modal detail popups and interactive toast notifications.## Core FeaturesDynamic Course Card Injection: Courses are dynamically managed via JavaScript object arrays and cleanly rendered onto the main content grid.Multi-Tiered Filtering System: * Main State Filters: Filter courses instantly by status (All, In Progress, Completed, Daily Goal).Category Sorting: Isolate courses belonging to Frontend, Backend, Data Science, or DevOps fields.Difficulty Pills: Quick toggles for Beginner, Intermediate, and Advanced skill levels.Live Interactive Search bar: An intuitive dropdown search box matching characters against titles, authors, or categories in real-time.Fully Accessible Modal UI: Clicking any card pops open an overlay with micro-statistics (total lessons, hours, remaining lessons) that is fully closeable via the Esc key or background clicks.Sleek Toast System: Instant feedback mechanisms built with custom-timed JavaScript intervals for clean action messages (e.g., download confirmations).Responsive layouts: Fluid adaptability that cleanly collapses columns down seamlessly on tablets and smartphones.## Technical Architecture & File StructuresThe project uses a standard, clean 3-layer architecture without requiring complex build steps or dependencies:### 1. Document Structure (index.html)The HTML markup establishes a semantic structure powered by semantic HTML5 tags:header.navbar: Houses global actions, user status badges (PRO LEARNER), notifications, and global lookup tools..layout Wrapper: A main grid controller separating the structural content layout.aside.sidebar: Contains navigation filters, vertical dropdowns, multi-pill difficulty filters, and premium upgrade graphics placeholders.main.main-content: Split into a prominent promotional tracking profile card (.hero-banner) and the main multi-column active interface layout (#coursesGrid).footer: Standard legal structures and design system annotations.### 2. Styling Core (style.css)Written entirely in CSS3 utilizing modern native structural layouts:CSS Custom Variables (:root): Keeps global colors, smooth transition curves (.22s cubic-bezier), typography sets (Inter & Space Grotesk), and depth profiles completely unified and dynamic.Advanced Layout Modules: Heavy reliance on custom dynamic layout systems (grid-template-columns) and modular distribution rules (flexbox).State-Driven Animations: Fine-tuned aesthetic keyframes such as an infinitely rotating logo icon indicator (@keyframes spin), smooth scale shifts on hover states, and dynamic backdrop blurring overlays (backdrop-filter).### 3.
